Open Finance Data Security Standard (OFDSS) updates framework, gains more supporters: anecdotes, Codat, Skyflow, and Very Good Security

August 2022

The Open Finance Data Security Standard (OFDSS) consortium is pleased to announce that four new companies have joined as supporters: anecdotes, Codat, Skyflow, and Very Good Security. These companies join existing supporters Flinks, MX, Plaid, Truework, Drata, Laika, Secureframe, and Vanta in backing the open standard for data security in digital finance.

About OFDSS

OFDSS was created as a collaborative framework for data security in emerging digital financial services. The standard establishes a common set of security requirements that are designed for modern, cloud-native companies operating in the financial services space.

The first draft of the standard was published in November 2021. Since then, the working committee has been refining the framework based on feedback from the industry.

What's New in Version 1.2

The latest version of OFDSS includes 79 individual security requirements across 13 control domains. The updated framework includes implementation guidelines and audit steps to help companies adopt the standard.

The 13 control domains cover:

  1. Resource Allocation
  2. Asset Management
  3. Access Controls
  4. Change Controls
  5. Software Development Life Cycle (SDLC)
  6. Cryptography
  7. Data Minimization
  8. Auditing and Alerting
  9. Incident Management
  10. Network Security
  11. Awareness and Training
  12. Vendor Management
  13. Independent Testing
